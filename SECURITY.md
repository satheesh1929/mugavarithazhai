🔐 Security Policy
📦 Supported Versions

We actively maintain and provide security updates for the following versions of our platform:

Version	Supported
Latest (Production)	✅
Previous Release (within 6 months)	✅
Older Versions	❌

⚠️ We recommend always using the latest version to ensure maximum security and performance.

🚨 Reporting a Vulnerability

We take security seriously. If you discover any vulnerability in our eCommerce platform, please report it responsibly.

📩 How to Report

Email us at: team.scode360@gmail.com

Or submit via: www.scode360.in

📝 What to Include

Please provide:

Description of the vulnerability

Steps to reproduce

Possible impact

Screenshots or proof-of-concept (if available)

⏱️ Response Time

Acknowledgement: Within 24–48 hours

Initial assessment: Within 3–5 business days

Fix & update: Based on severity (usually within 7–14 days)

✅ What to Expect

If the issue is valid, we will:

Fix the vulnerability as soon as possible

Keep you updated on progress

Credit you (optional) for responsible disclosure

If the issue is declined:

We will provide a clear explanation

🔒 Responsible Disclosure Guidelines

Do NOT exploit the vulnerability for personal gain

Do NOT access or modify user data

Do NOT perform actions that may harm the platform or users

🙏 Acknowledgment

We appreciate the efforts of security researchers and users who help us keep our platform safe.
