# Mugavari Thazhai - E-commerce Website

This is a static e-commerce website for Mugavari Thazhai, featuring authentic products from Pollachi.

## How to Run

Since this is a static website using HTML and Tailwind CSS (via CDN), you do **not** need `npm` or `node.js` to run it.

1.  Simply locate the folder `Mugavari Thazhai` on your desktop.
2.  Double-click `index.html` to open it in your web browser.
3.  Navigate to the **Products** page using the menu links.

## Technologies Used

-   **HTML5**: Semantic structure.
-   **Tailwind CSS (CDN)**: Modern, responsive styling.
-   **JavaScript**: Simple interactions (Mobile menu, Cart logic, WhatsApp ordering).
-   **FontAwesome**: Icons.
-   **Google Fonts**: "Poppins" typeface.

## Features

-   Responsive Design (Mobile & Desktop).
-   Product Listing with Cart functionality.
-   WhatsApp Ordering System.
-   Premium "Nature" Aesthetic.

> **Note**: If you want to use a local development server, you can install the "Live Server" extension in VS Code.
